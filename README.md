## 🛠️ Configuração

O deploy automático está configurado através do GitHub Actions. Qualquer push para a branch `main` irá automaticamente publicar o site.

### Requisitos para GitHub Pages estar ativo:

1. ✅ Repositório com conteúdo público
2. ✅ Workflow de deploy configurado (`.github/workflows/pages.yml`)
3. ⚠️ **IMPORTANTE**: Você deve ir para **Settings > Pages** do repositório e:
   - Selecionar "GitHub Actions" como source
   - Ou deixar no modo automático
